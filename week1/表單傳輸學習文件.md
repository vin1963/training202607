# JSP 表單傳輸學習文件 - Checkbox & Radio

## 學習目標

1. 理解 HTML 表單中 Radio 與 Checkbox 的差異
2. 掌握 `param` 與 `paramValues` 的使用時機
3. 學會 EL 與 Scriptlet 兩種讀取參數的方式
4. 了解字串與數值的轉換處理

---

## 1. 表單元素基礎

### 1.1 Radio（單選按鈕）

```html
<input type="radio" name="gender" value="male"> 男性
<input type="radio" name="gender" value="female"> 女性
<input type="radio" name="gender" value="other"> 其他
```

**特性：**
- `name` 必須相同才能實現互斥選擇
- 只會送出一個值
- 使用 `${param.name}` 讀取

### 1.2 Checkbox（複選框）

```html
<input type="checkbox" name="hobbies" value="reading"> 閱讀
<input type="checkbox" name="hobbies" value="sports"> 運動
<input type="checkbox" name="hobbies" value="music"> 音樂
```

**特性：**
- `name` 相同但可選多個
- 可能送出多個值（陣列）
- 使用 `${paramValues.name}` 讀取

---

## 2. 參數讀取方式比較

### 2.1 讀取單一值（Radio / Text）

| 方式 | 語法 | 說明 |
|------|------|------|
| EL | `${param.name}` | 簡潔，自動處理 null |
| Scriptlet | `request.getParameter("name")` | 需手動處理 null |

### 2.2 讀取多值（Checkbox）

| 方式 | 語法 | 說明 |
|------|------|------|
| EL | `${paramValues.name[0]}` | 依索引取值 |
| Scriptlet | `request.getParameterValues("name")` | 回傳 String 陣列 |

## 3. 程式碼解析

### 3.1 form.jsp - 表單頁面

```jsp
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<%@ taglib prefix="c" uri="jakarta.tags.core" %>
```

**重點說明：**
- `contentType`：設定回應的 MIME 類型與編碼
- `taglib`：引入 JSTL 標籤庫，`prefix="c"` 代表核心標籤

#### 表單屬性說明

```html
<form action="result.jsp" method="post">
```

| 屬性 | 值 | 說明 |
|------|-----|------|
| action | result.jsp | 送出後導向的頁面 |
| method | post | 以 POST 方式傳送（資料不會顯示在 URL） |

### 3.2 result.jsp - 結果頁面

#### 方法一：EL 表達式（簡單值）

```jsp
<%-- 讀取文字輸入 --%>
<p>姓名：${param.name}</p>

<%-- 讀取 Radio，搭配三元運算子 --%>
<p>性別：${param.gender == 'male' ? '男性' : (param.gender == 'female' ? '女性' : '其他')}</p>
```

#### 方法二：Scriptlet（複雜邏輯）

```jsp
<%
    // 讀取 Checkbox 多值
    String[] hobbies = request.getParameterValues("hobbies");
    if (hobbies != null) {
        for (int i = 0; i < hobbies.length; i++) {
            String hobby = hobbies[i];
            if (hobby.equals("reading")) out.print("閱讀");
            else if (hobby.equals("sports")) out.print("運動");
            else if (hobby.equals("music")) out.print("音樂");
            else if (hobby.equals("travel")) out.print("旅行");
            else if (hobby.equals("gaming")) out.print("遊戲");
            if (i < hobbies.length - 1) out.print("、");
        }
    } else {
        out.print("未選擇");
    }
%>
```

#### 方法三：Scriptlet（數值比較）

```jsp
<%
    String a = request.getParameter("va");
    String b = request.getParameter("vb");
    if (a != null && b != null) {
        int va = Integer.parseInt(a);  // 字串轉整數
        int vb = Integer.parseInt(b);
        if (va > vb) {
            out.print(va + " > " + vb);
        } else if (va < vb) {
            out.print(va + " < " + vb);
        } else {
            out.print(va + " == " + vb);
        }
    } else {
        out.print("缺少參數");
    }
%>
```

## 4. EL 運算子注意事項

### 4.1 字串與數值比較

```jsp
<%-- ❌ 錯誤：param 是字串，做字典序比較 --%>
${param.va > param.vb}

<%-- ✅ 正確：轉為數值後比較 --%>
${Integer.parseInt(param.va) > Integer.parseInt(param.vb)}
```

### 4.2 特殊字元處理

```jsp
<%-- ❌ 錯誤：> 和 < 是 EL 運算子 --%>
${param.va + ">" + param.vb}

<%-- ✅ 正確：使用 HTML 實體 --%>
${param.va + "&gt;" + param.vb}
${param.va + "&lt;" + param.vb}

<%-- ✅ 或使用 EL 字串串接 --%>
${param.va}${'>'}${param.vb}
```

## 5. EL vs Scriptlet 比較

| 特性 | EL | Scriptlet |
|------|-----|-----------|
| 語法 | `${expression}` | `<% code %>` |
| 可讀性 | 高 | 中 |
| 功能 | 簡單取值、運算 | 完整 Java 邏輯 |
| null 處理 | 自動 | 需手動 |
| 適用場景 | 顯示資料 | 複雜處理 |

**建議：**
- 簡單取值、條件判斷 → 使用 **EL**
- 迴圈、例外處理、複雜邏輯 → 使用 **Scriptlet**

## 8. 常見錯誤

| 錯誤 | 原因 | 解決方法 |
|------|------|----------|
| `null` 顯示 | 未勾選的 checkbox 不會送出值 | 先判斷 `empty paramValues.xxx` |
| `NumberFormatException` | 字串無法轉為數值 | 確認參數存在且格式正確 |
| EL 比較錯誤 | 參數是字串型別 | 使用 `Integer.parseInt()` 轉型 |
| 特殊字元解析錯誤 | `>` `<` 是 EL 運算子 | 使用 `&gt;` `&lt;` 或分開串接 |
