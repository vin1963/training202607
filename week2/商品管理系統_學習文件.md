# 題目一：商品管理系統（基礎 CRUD）— 學習說明文件

## 一、功能概述

商品管理系統實作了完整的 CRUD 操作，包含以下功能：

| 功能 | HTTP 請求 | 對應 DAO 方法 | SQL 操作 |
|---|---|---|---|
| 列表查詢 | GET /product/list | `findAll()` | SELECT (含軟刪除過濾) |
| 新增表單 | GET /product/add | — | — |
| 編輯表單 | GET /product/edit?id=X | `findById(id)` | SELECT WHERE id = ? |
| 刪除 | GET /product/delete?id=X | `delete(id)` | UPDATE SET deleted = 1 |
| 新增提交 | POST /product | `insert(product)` | INSERT + RETURN_GENERATED_KEYS |
| 編輯提交 | POST /product (含 id) | `update(product)` | UPDATE SET ... WHERE id = ? |
| 搜尋 | GET /product/search | `searchByName` / `findByCategory` | SELECT ... LIKE / WHERE category = ? |

## 二、MVC 架構對應

```
瀏覽器請求
    │
    ▼
ProductServlet (Controller)
    │  doGet() / doPost() 解析請求路徑
    │  呼叫 ProductDao 方法
    │  設定 request.setAttribute()
    │  轉發或重導向
    │
    ├──→ ProductDao (Model - 介面)
    │       定義 CRUD contract
    │
    ├──→ ProductDaoImpl (Model - 實作)
    │       JDBC 操作 MySQL
    │       PreparedStatement 防 SQL Injection
    │       try-with-resources 自動釋放資源
    │
    └──→ productList.jsp / productForm.jsp (View)
            EL + JSTL 呈現資料
            無 Java 程式碼
```

## 三、Model 層：Product.java (JavaBean)

```java
package com.example.model;

import java.math.BigDecimal;

public class Product {
    private int id;              // 商品編號 (自動生成)
    private String name;         // 商品名稱
    private BigDecimal price;    // 商品價格 (精確十進位)
    private int stock;           // 庫存數量
    private String category;     // 商品類別
    private boolean deleted;     // 軟刪除標記 (0:正常, 1:已刪除)

    // 無參數建構子 (JSP EL / 框架反射需要)
    public Product() {
    }

    // 新增用建構子 (不含 id，由 DB 自動生成)
    public Product(String name, BigDecimal price, int stock, String category) {
        this.name = name;
        this.price = price;
        this.stock = stock;
        this.category = category;
    }

    // 完整建構子 (查詢 / 編輯時使用)
    public Product(int id, String name, BigDecimal price, int stock, String category, boolean deleted) {
        this.id = id;
        this.name = name;
        this.price = price;
        this.stock = stock;
        this.category = category;
        this.deleted = deleted;
    }

    // --- Getters & Setters ---

    public int getId() {
        return id;
    }

    public void setId(int id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public BigDecimal getPrice() {
        return price;
    }

    public void setPrice(BigDecimal price) {
        this.price = price;
    }

    public int getStock() {
        return stock;
    }

    public void setStock(int stock) {
        this.stock = stock;
    }

    public String getCategory() {
        return category;
    }

    public void setCategory(String category) {
        this.category = category;
    }

    public boolean isDeleted() {
        return deleted;
    }

    public void setDeleted(boolean deleted) {
        this.deleted = deleted;
    }

    @Override
    public String toString() {
        return "Product{id=" + id + ", name='" + name + "', price=" + price
                + ", stock=" + stock + ", category='" + category + "'}";
    }
}
```

**重點說明：**
- `price` 使用 `BigDecimal` 而非 `double` / `float`，避免浮點數精度誤差
- 屬性命名與資料表欄位一致，便於 `mapRow()` 對映
- 提供無參數建構子（JSP EL 存取需要）與有參數建構子（方便建立）
- `toString()` 方便除錯時快速檢視物件內容

## 四、DAO 層

### 4.1 ProductDao 介面（Contract）

```java
package com.example.dao;

import com.example.model.Product;
import java.util.List;
import java.util.Optional;

public interface ProductDao {
    void insert(Product product);                   // C
    Optional<Product> findById(int id);             // R (單筆)
    List<Product> findAll();                        // R (多筆)
    List<Product> searchByName(String name);        // R (模糊)
    List<Product> findByCategory(String category);  // R (類別)
    void update(Product product);                   // U
    void delete(int id);                            // D (軟刪除)
    long count();                                   // 總筆數 (分頁用)
}
```

**設計原則：**
- 先定義介面再實作，Controller 只依賴 `ProductDao`，不依賴實作類別
- `findById` 回傳 `Optional<Product>` 明確表達可能查無資料
- `count()` 提供給分頁使用

### 4.2 ProductDaoImpl 完整程式碼

以下是將所有 CRUD 片段組裝成一個完整、可編譯的類別：

```java
package com.example.dao.impl;

import com.example.dao.ProductDao;
import com.example.model.Product;
import com.example.util.DBUtil;

import java.math.BigDecimal;
import java.sql.*;
import java.util.ArrayList;
import java.util.List;
import java.util.Optional;

public class ProductDaoImpl implements ProductDao {

    // ========== CREATE — INSERT ==========
    @Override
    public void insert(Product product) {
        String sql = "INSERT INTO product (name, price, stock, category) VALUES (?, ?, ?, ?)";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql, Statement.RETURN_GENERATED_KEYS)) {
            ps.setString(1, product.getName());
            ps.setBigDecimal(2, product.getPrice());
            ps.setInt(3, product.getStock());
            ps.setString(4, product.getCategory());
            ps.executeUpdate();
            // 取得自動產生的 ID 並寫回 JavaBean
            try (ResultSet rs = ps.getGeneratedKeys()) {
                if (rs.next()) {
                    product.setId(rs.getInt(1));
                }
            }
        } catch (SQLException e) {
            throw new RuntimeException("新增商品失敗", e);
        }
    }

    // ========== READ — SELECT (單筆) ==========
    @Override
    public Optional<Product> findById(int id) {
        String sql = "SELECT * FROM product WHERE id = ? AND deleted = 0";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setInt(1, id);
            try (ResultSet rs = ps.executeQuery()) {
                if (rs.next()) {
                    return Optional.of(mapRow(rs));
                }
            }
        } catch (SQLException e) {
            throw new RuntimeException("查詢商品失敗, id=" + id, e);
        }
        return Optional.empty();
    }

    // ========== READ — SELECT (多筆) ==========
    @Override
    public List<Product> findAll() {
        String sql = "SELECT * FROM product WHERE deleted = 0";
        List<Product> list = new ArrayList<>();
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql);
             ResultSet rs = ps.executeQuery()) {
            while (rs.next()) {
                list.add(mapRow(rs));
            }
        } catch (SQLException e) {
            throw new RuntimeException("查詢所有商品失敗", e);
        }
        return list;
    }

    // ========== READ — 模糊搜尋 ==========
    @Override
    public List<Product> searchByName(String name) {
        String sql = "SELECT * FROM product WHERE deleted = 0 AND name LIKE ?";
        List<Product> list = new ArrayList<>();
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setString(1, "%" + name + "%");
            try (ResultSet rs = ps.executeQuery()) {
                while (rs.next()) {
                    list.add(mapRow(rs));
                }
            }
        } catch (SQLException e) {
            throw new RuntimeException("搜尋商品失敗, name=" + name, e);
        }
        return list;
    }

    // ========== READ — 依類別查詢 ==========
    @Override
    public List<Product> findByCategory(String category) {
        String sql = "SELECT * FROM product WHERE deleted = 0 AND category = ?";
        List<Product> list = new ArrayList<>();
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setString(1, category);
            try (ResultSet rs = ps.executeQuery()) {
                while (rs.next()) {
                    list.add(mapRow(rs));
                }
            }
        } catch (SQLException e) {
            throw new RuntimeException("依類別查詢失敗, category=" + category, e);
        }
        return list;
    }

    // ========== UPDATE ==========
    @Override
    public void update(Product product) {
        String sql = "UPDATE product SET name = ?, price = ?, stock = ?, category = ? "
                   + "WHERE id = ? AND deleted = 0";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setString(1, product.getName());
            ps.setBigDecimal(2, product.getPrice());
            ps.setInt(3, product.getStock());
            ps.setString(4, product.getCategory());
            ps.setInt(5, product.getId());
            ps.executeUpdate();
        } catch (SQLException e) {
            throw new RuntimeException("更新商品失敗, id=" + product.getId(), e);
        }
    }

    // ========== DELETE — 軟刪除 ==========
    @Override
    public void delete(int id) {
        String sql = "UPDATE product SET deleted = 1 WHERE id = ?";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setInt(1, id);
            ps.executeUpdate();
        } catch (SQLException e) {
            throw new RuntimeException("刪除商品失敗, id=" + id, e);
        }
    }

    // ========== COUNT — 總筆數 ==========
    @Override
    public long count() {
        String sql = "SELECT COUNT(*) FROM product WHERE deleted = 0";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql);
             ResultSet rs = ps.executeQuery()) {
            if (rs.next()) {
                return rs.getLong(1);
            }
        } catch (SQLException e) {
            throw new RuntimeException("計算商品總數失敗", e);
        }
        return 0;
    }

    // ========== mapRow — ResultSet 轉 JavaBean ==========
    private Product mapRow(ResultSet rs) throws SQLException {
        Product product = new Product();
        product.setId(rs.getInt("id"));
        product.setName(rs.getString("name"));
        product.setPrice(rs.getBigDecimal("price"));
        product.setStock(rs.getInt("stock"));
        product.setCategory(rs.getString("category"));
        product.setDeleted(rs.getBoolean("deleted"));
        return product;
    }
}
```

**各方法對照說明：**

| 方法 | SQL | 重點 |
|---|---|---|
| `insert()` | `INSERT INTO ... RETURN_GENERATED_KEYS` | `getGeneratedKeys()` 取得自增 ID 回寫物件 |
| `findById()` | `SELECT ... WHERE id = ? AND deleted = 0` | `Optional.of()` / `Optional.empty()` 表達有無資料 |
| `findAll()` | `SELECT * FROM product WHERE deleted = 0` | `while (rs.next())` 迴圈收集多筆 |
| `searchByName()` | `... WHERE name LIKE ?` | `"%" + name + "%"` 由 PreparedStatement 設定，防 SQL Injection |
| `findByCategory()` | `... WHERE category = ?` | 精確比對類別 |
| `update()` | `UPDATE ... WHERE id = ? AND deleted = 0` | `AND deleted = 0` 避免更新已刪除資料 |
| `delete()` | `UPDATE SET deleted = 1 WHERE id = ?` | 軟刪除，資料保留可復原 |
| `count()` | `SELECT COUNT(*)` | 提供分頁使用 |
| `mapRow()` | — | 私有方法， ResultSet → JavaBean 對映 |

**為什麼用軟刪除而不是 DELETE？**

| | 軟刪除 | 物理刪除 |
|---|---|---|
| SQL | `UPDATE SET deleted = 1` | `DELETE FROM ...` |
| 資料保留 | 保留 | 永久消失 |
| 可復原 | 可 | 不可 |
| 查詢 | 需加 `WHERE deleted = 0` | 不需過濾 |
| 適合場景 | 多數商業系統 | 暫存或測試資料 |

## 五、Controller 層：ProductServlet.java

### 5.1 完整程式碼

```java
package com.example.servlet;

import com.example.dao.ProductDao;
import com.example.dao.impl.ProductDaoImpl;
import com.example.model.Product;

import jakarta.servlet.ServletException;
import jakarta.servlet.annotation.WebServlet;
import jakarta.servlet.http.HttpServlet;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;

import java.io.IOException;
import java.math.BigDecimal;
import java.util.List;
import java.util.Optional;

@WebServlet("/product/*")
public class ProductServlet extends HttpServlet {

    private final ProductDao productDao = new ProductDaoImpl();

    // ========== GET 請求處理 ==========
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {

        String path = req.getPathInfo();  // 取得 /product 之後的路徑，如 /list, /edit

        if (path == null) {
            path = "/";
        }

        switch (path) {
            case "/", "/list" -> handleList(req, resp);
            case "/add"       -> handleAddForm(req, resp);
            case "/edit"      -> handleEditForm(req, resp);
            case "/delete"    -> handleDelete(req, resp);
            case "/search"    -> handleSearch(req, resp);
            default           -> resp.sendError(HttpServletResponse.SC_NOT_FOUND, "找不到頁面: " + path);
        }
    }

    // ========== POST 請求處理 ==========
    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {

        req.setCharacterEncoding("UTF-8");

        String idParam = req.getParameter("id");
        String name = req.getParameter("name");
        String priceStr = req.getParameter("price");
        String stockStr = req.getParameter("stock");
        String category = req.getParameter("category");

        // --- 後端資料驗證 ---
        if (name == null || name.trim().isEmpty()) {
            req.setAttribute("error", "商品名稱不可為空");
            forwardToForm(req, resp, null);
            return;
        }

        BigDecimal price;
        try {
            price = new BigDecimal(priceStr);
            if (price.compareTo(BigDecimal.ZERO) <= 0) {
                req.setAttribute("error", "價格必須大於 0");
                forwardToForm(req, resp, null);
                return;
            }
        } catch (NumberFormatException | NullPointerException e) {
            req.setAttribute("error", "價格格式不正確");
            forwardToForm(req, resp, null);
            return;
        }

        int stock;
        try {
            stock = Integer.parseInt(stockStr);
            if (stock < 0) {
                req.setAttribute("error", "庫存不可為負數");
                forwardToForm(req, resp, null);
                return;
            }
        } catch (NumberFormatException | NullPointerException e) {
            req.setAttribute("error", "庫存格式不正確");
            forwardToForm(req, resp, null);
            return;
        }

        if (category == null) {
            category = "";
        }

        // --- 根據有無 id 決定新增或更新 ---
        if (idParam == null || idParam.isEmpty()) {
            // 新增模式
            Product product = new Product(name.trim(), price, stock, category.trim());
            productDao.insert(product);
        } else {
            // 編輯模式
            Product product = new Product();
            product.setId(Integer.parseInt(idParam));
            product.setName(name.trim());
            product.setPrice(price);
            product.setStock(stock);
            product.setCategory(category.trim());
            productDao.update(product);
        }

        // PRG 模式：Post-Redirect-Get，避免重複提交
        resp.sendRedirect(req.getContextPath() + "/product/list");
    }

    // ========== 各處理方法 ==========

    // 列表查詢
    private void handleList(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        List<Product> products = productDao.findAll();
        req.setAttribute("products", products);
        req.getRequestDispatcher("/WEB-INF/views/productList.jsp").forward(req, resp);
    }

    // 新增表單
    private void handleAddForm(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        // 不帶 product 物件，JSP 判斷為新增模式
        req.getRequestDispatcher("/WEB-INF/views/productForm.jsp").forward(req, resp);
    }

    // 編輯表單
    private void handleEditForm(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        String idStr = req.getParameter("id");
        if (idStr == null || idStr.isEmpty()) {
            resp.sendRedirect(req.getContextPath() + "/product/list");
            return;
        }
        int id = Integer.parseInt(idStr);
        Optional<Product> opt = productDao.findById(id);
        if (opt.isPresent()) {
            req.setAttribute("product", opt.get());
            req.getRequestDispatcher("/WEB-INF/views/productForm.jsp").forward(req, resp);
        } else {
            resp.sendRedirect(req.getContextPath() + "/product/list");
        }
    }

    // 刪除（軟刪除）
    private void handleDelete(HttpServletRequest req, HttpServletResponse resp)
            throws IOException {
        String idStr = req.getParameter("id");
        if (idStr != null && !idStr.isEmpty()) {
            productDao.delete(Integer.parseInt(idStr));
        }
        resp.sendRedirect(req.getContextPath() + "/product/list");
    }

    // 搜尋
    private void handleSearch(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        String name = req.getParameter("name");
        String category = req.getParameter("category");
        List<Product> products;

        if (name != null && !name.trim().isEmpty()) {
            products = productDao.searchByName(name.trim());
        } else if (category != null && !category.trim().isEmpty()) {
            products = productDao.findByCategory(category.trim());
        } else {
            products = productDao.findAll();
        }

        req.setAttribute("products", products);
        req.getRequestDispatcher("/WEB-INF/views/productList.jsp").forward(req, resp);
    }

    // 轉發表單（驗證失敗時回帶錯誤訊息）
    private void forwardToForm(HttpServletRequest req, HttpServletResponse resp,
                               Product product) throws ServletException, IOException {
        if (product != null) {
            req.setAttribute("product", product);
        }
        req.getRequestDispatcher("/WEB-INF/views/productForm.jsp").forward(req, resp);
    }
}
```

**路由分發對照：**

| GET 路徑 | 處理方法 | 行為 |
|---|---|---|
| `/product/list` 或 `/product/` | `handleList()` | 查詢所有 → 轉發 productList.jsp |
| `/product/add` | `handleAddForm()` | 轉發表單（空表單，新增模式） |
| `/product/edit?id=X` | `handleEditForm()` | 查詢單筆 → 轉發表單（編輯模式） |
| `/product/delete?id=X` | `handleDelete()` | 軟刪除 → 重導向列表 |
| `/product/search?name=X` | `handleSearch()` | 模糊搜尋 → 轉發列表 |

**PRG 模式 (Post-Redirect-Get)：** 新增/更新完成後使用 `sendRedirect` 而非 `forward`，避免使用者重新整理頁面時重複提交表單。

### 5.2 三層資料驗證

| 層級 | 位置 | 驗證內容 | 做法 |
|---|---|---|---|
| 前端 | productForm.jsp | 非空、格式 | `required` 屬性、`type="number"` |
| 後端 | ProductServlet.java | 名稱非空、價格 > 0、庫存 ≥ 0 | 條件判斷，不符則返回表單並顯示錯誤 |
| 資料庫 | MySQL | 非空、唯一 | 欄位 `NOT NULL`、`UNIQUE` |

| 情境 | JSP 寫法 | 說明 |

## 十、EncodingFilter.java — 完整程式碼

```java
package com.example.filter;

import jakarta.servlet.*;
import jakarta.servlet.annotation.WebFilter;
import java.io.IOException;

@WebFilter("/*")
public class EncodingFilter implements Filter {

    @Override
    public void init(FilterConfig filterConfig) throws ServletException {
        // 可從 filterConfig 讀取初始化參數（此例不需）
    }

    @Override
    public void doFilter(ServletRequest request, ServletResponse response,
                         FilterChain chain) throws IOException, ServletException {
        request.setCharacterEncoding("UTF-8");
        response.setCharacterEncoding("UTF-8");
        response.setContentType("text/html;charset=UTF-8");
        chain.doFilter(request, response);
    }

    @Override
    public void destroy() {
        // 釋放資源（此例不需）
    }
}
```

**為什麼需要 Filter？** 若不設定編碼，`req.getParameter("name")` 取得到的中文會是亂碼。`@WebFilter("/*")` 表示所有請求都經過此 Filter。

**Filter 生命週期：**

| 方法 | 時機 | 用途 |
|---|---|---|
| `init()` | Web 應用啟動時執行一次 | 讀取初始化參數 |
| `doFilter()` | 每個請求都會執行 | 設定編碼、呼叫下一個 Filter |
| `destroy()` | Web 應用關閉時執行一次 | 釋放資源 |

## 十三、軟刪除 (Soft Delete) 模式

### 資料表結構

```sql
CREATE TABLE product (
    id      INT AUTO_INCREMENT PRIMARY KEY,
    name    VARCHAR(100) NOT NULL,
    price   DECIMAL(10,2) NOT NULL,
    stock   INT NOT NULL DEFAULT 0,
    category VARCHAR(50),
    deleted TINYINT DEFAULT 0     -- 0: 正常, 1: 已刪除
);
```

### 對 DAO 的影響

| 操作 | 一般寫法 | 軟刪除寫法 |
|---|---|---|
| 新增 | 不變 | 不變（deleted 預設 0） |
| 查詢 | `SELECT * FROM product` | `SELECT * FROM product WHERE deleted = 0` |
| 修改 | `UPDATE ... WHERE id = ?` | `UPDATE ... WHERE id = ? AND deleted = 0` |
| 刪除 | `DELETE FROM product WHERE id = ?` | `UPDATE product SET deleted = 1 WHERE id = ?` |

## 十四、PreparedStatement 防 SQL Injection

### ❌ 錯誤寫法（字串拼接）

```java
String sql = "SELECT * FROM product WHERE name LIKE '%" + name + "%'";
```

使用者輸入 `' OR 1=1 --` 時，SQL 變成：
```sql
SELECT * FROM product WHERE name LIKE '%' OR 1=1 --%'
```
**所有資料都被查出，安全性漏洞。**

### ✅ 正確寫法（使用 ? 佔位符）

```java
String sql = "SELECT * FROM product WHERE name LIKE ?";
ps.setString(1, "%" + name + "%");
```

`PreparedStatement` 會自動跳脫特殊字元，`?` 的位置只會被當作「值」而非「SQL 語法」。

## 十五、try-with-resources 資源管理

```java
try (Connection conn = DBUtil.getConnection();
     PreparedStatement ps = conn.prepareStatement(sql);
     ResultSet rs = ps.executeQuery()) {
    // 使用 conn, ps, rs
} catch (SQLException e) {
    throw new RuntimeException(e);
}
// 離開 try 區塊時，rs, ps, conn 會自動關閉 (AutoCloseable)
```

**對比舊寫法：** 以前需要在 `finally` 區塊中逐一 `close()`，容易漏掉導致連線泄漏。

## 十七、常見錯誤排解

| 錯誤現象 | 原因 | 解決方式 |
|---|---|---|
| `ClassNotFoundException: com.mysql.cj.jdbc.Driver` | MySQL Connector 未加入 | 確認 pom.xml 有 `mysql-connector-j` |
| 500 JasperException | JSTL jar 缺失 | 確認 JSTL API + Implementation 依賴 |
| 404 | URL 路徑錯誤 | 確認 `@WebServlet("/product/*")` 與 JSP 中 `contextPath` 一致 |
| 中文變成 `???` | 編碼未設定 | 確認 EncodingFilter 存在且正確 |
| `Table 'practice_db.product' doesn't exist` | 資料表未建立 | 執行 SQL 建表 |
| `Data too long for column 'name'` | 資料長度超過欄位限制 | 調整 VARCHAR 長度或檢查輸入 |
| `Duplicate entry for key` | 違反 UNIQUE 約束 | 檢查資料是否重複 |
