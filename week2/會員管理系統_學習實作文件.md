# 題目二：會員管理系統（CRUD + 登入登出）— 學習實作文件

## 一、功能概述

本專案在商品管理系統的 MVC + DAO 架構上，新增 **Session 管理** 與 **LoginFilter** 兩個核心概念。

| 功能 | HTTP 請求 | 對應方法 | SQL 操作 |
|---|---|---|---|
| 登入頁面 | GET /member/login | `handleLoginPage()` | — |
| 登入驗證 | POST /member/login | `handleLogin()` | SELECT WHERE account=? AND password=? |
| 登出 | GET /member/logout | `handleLogout()` | — |
| 列表查詢 | GET /member/list | `handleList()` | SELECT (含軟刪除過濾) |
| 新增表單 | GET /member/add | `handleAddForm()` | — |
| 編輯表單 | GET /member/edit?id=X | `handleEditForm()` | SELECT WHERE id=? |
| 刪除 | GET /member/delete?id=X | `handleDelete()` | UPDATE SET deleted=1 |
| 新增提交 | POST /member | `handleCreate()` | INSERT + RETURN_GENERATED_KEYS |
| 編輯提交 | POST /member (含 id) | `handleUpdate()` | UPDATE SET ... WHERE id=? |
| 搜尋 | GET /member/search | `handleSearch()` | SELECT ... LIKE / WHERE role=? |

## 二、與商品管理系統的差異（本專案學習重點）

| 新概念 | 說明 | 對應檔案 |
|---|---|---|
| **Session** | 伺服器端保存使用者登入狀態 | MemberServlet.java |
| **登入/登出** | 驗證帳號密碼，建立/销毁 Session | MemberServlet.java |
| **LoginFilter** | 未登入時自動導向登入頁 | LoginFilter.java |
| **密碼比對** | 文字比對（學習階段，正式專案應雜湊） | MemberDaoImpl.java |
| **Session 保護** | 只有登入使用者才能存取會員資料 | LoginFilter.java |

### 架構圖

```
瀏覽器請求
    │
    ▼
LoginFilter (Session 檢查)
    │  未登入 → 導向 /member/login
    │  已登入 → 放行
    │
    ▼
MemberServlet (Controller)
    │  doGet() / doPost() 解析請求路徑 (switch)
    │  呼叫 MemberDao 方法
    │  設定 request.setAttribute() / session.setAttribute()
    │  轉發或重導向
    │
    ├──→ MemberDao (Model - 介面)
    │       定義 CRUD contract
    │
    ├──→ MemberDaoImpl (Model - 實作)
    │       JDBC 操作 MySQL
    │
    └──→ login.jsp / memberList.jsp / memberForm.jsp (View)
            EL + JSTL 呈現資料
```

```java
package com.example.model;

public class Member {
    private int id;              // 會員編號 (自動生成)
    private String account;      // 帳號 (登入用，唯一)
    private String password;     // 密碼
    private String name;         // 姓名
    private String email;        // 信箱
    private String phone;        // 電話
    private String role;         // 角色: admin / user
    private boolean deleted;     // 軟刪除標記

    // 無參數建構子 (JSP EL / 框架反射需要)
    public Member() {
    }

    // 新增用建構子 (不含 id)
    public Member(String account, String password, String name,
                  String email, String phone, String role) {
        this.account = account;
        this.password = password;
        this.name = name;
        this.email = email;
        this.phone = phone;
        this.role = role;
    }

    // --- Getters & Setters ---

    public int getId() { return id; }
    public void setId(int id) { this.id = id; }

    public String getAccount() { return account; }
    public void setAccount(String account) { this.account = account; }

    public String getPassword() { return password; }
    public void setPassword(String password) { this.password = password; }

    public String getName() { return name; }
    public void setName(String name) { this.name = name; }

    public String getEmail() { return email; }
    public void setEmail(String email) { this.email = email; }

    public String getPhone() { return phone; }
    public void setPhone(String phone) { this.phone = phone; }

    public String getRole() { return role; }
    public void setRole(String role) { this.role = role; }

    public boolean isDeleted() { return deleted; }
    public void setDeleted(boolean deleted) { this.deleted = deleted; }

    @Override
    public String toString() {
        return "Member{id=" + id + ", account='" + account + "', name='" + name
                + "', role='" + role + "'}";
    }
}
```

**重點說明：**
- `account` 設定 `UNIQUE` 約束，登入時以帳號查詢
- `role` 區分管理者與一般使用者（本例不實作權限控僅，僅做查詢標記）
- `password` 直接儲存文字（學習階段），正式專案應使用 BCrypt 雜湊

### 7.1 MemberDao 介面（Contract）

```java
package com.example.dao;

import com.example.model.Member;
import java.util.List;
import java.util.Optional;

public interface MemberDao {
    void insert(Member member);                      // C
    Optional<Member> findById(int id);               // R (單筆)
    Optional<Member> findByAccount(String account);   // R (依帳號查詢，登入用)
    List<Member> findAll();                          // R (多筆)
    List<Member> searchByName(String name);          // R (模糊搜尋)
    List<Member> findByRole(String role);             // R (依角色)
    void update(Member member);                      // U
    void delete(int id);                             // D (軟刪除)
    long count();                                    // 總筆數
}
```

**與 ProductDao 的差異：**
- 新增 `findByAccount(String account)` — 登入驗證時需要用帳號查會員
- 其餘 CRUD 方法與 ProductDao 結構相同

### 7.2 MemberDaoImpl 完整程式碼

```java
package com.example.dao.impl;

import com.example.dao.MemberDao;
import com.example.model.Member;
import com.example.util.DBUtil;

import java.sql.*;
import java.util.ArrayList;
import java.util.List;
import java.util.Optional;

public class MemberDaoImpl implements MemberDao {

    // ========== CREATE — INSERT ==========
    @Override
    public void insert(Member member) {
        String sql = "INSERT INTO member (account, password, name, email, phone, role) "
                   + "VALUES (?, ?, ?, ?, ?, ?)";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql, Statement.RETURN_GENERATED_KEYS)) {
            ps.setString(1, member.getAccount());
            ps.setString(2, member.getPassword());
            ps.setString(3, member.getName());
            ps.setString(4, member.getEmail());
            ps.setString(5, member.getPhone());
            ps.setString(6, member.getRole());
            ps.executeUpdate();
            try (ResultSet rs = ps.getGeneratedKeys()) {
                if (rs.next()) {
                    member.setId(rs.getInt(1));
                }
            }
        } catch (SQLException e) {
            throw new RuntimeException("新增會員失敗", e);
        }
    }

    // ========== READ — 依帳號查詢 (登入用) ==========
    @Override
    public Optional<Member> findByAccount(String account) {
        String sql = "SELECT * FROM member WHERE account = ? AND deleted = 0";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setString(1, account);
            try (ResultSet rs = ps.executeQuery()) {
                if (rs.next()) {
                    return Optional.of(mapRow(rs));
                }
            }
        } catch (SQLException e) {
            throw new RuntimeException("依帳號查詢失敗, account=" + account, e);
        }
        return Optional.empty();
    }

    // ========== READ — SELECT (單筆) ==========
    @Override
    public Optional<Member> findById(int id) {
        String sql = "SELECT * FROM member WHERE id = ? AND deleted = 0";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setInt(1, id);
            try (ResultSet rs = ps.executeQuery()) {
                if (rs.next()) {
                    return Optional.of(mapRow(rs));
                }
            }
        } catch (SQLException e) {
            throw new RuntimeException("查詢會員失敗, id=" + id, e);
        }
        return Optional.empty();
    }

    // ========== READ — SELECT (多筆) ==========
    @Override
    public List<Member> findAll() {
        String sql = "SELECT * FROM member WHERE deleted = 0";
        List<Member> list = new ArrayList<>();
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql);
             ResultSet rs = ps.executeQuery()) {
            while (rs.next()) {
                list.add(mapRow(rs));
            }
        } catch (SQLException e) {
            throw new RuntimeException("查詢所有會員失敗", e);
        }
        return list;
    }

    // ========== READ — 模糊搜尋 ==========
    @Override
    public List<Member> searchByName(String name) {
        String sql = "SELECT * FROM member WHERE deleted = 0 AND name LIKE ?";
        List<Member> list = new ArrayList<>();
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setString(1, "%" + name + "%");
            try (ResultSet rs = ps.executeQuery()) {
                while (rs.next()) {
                    list.add(mapRow(rs));
                }
            }
        } catch (SQLException e) {
            throw new RuntimeException("搜尋會員失敗, name=" + name, e);
        }
        return list;
    }

    // ========== READ — 依角色查詢 ==========
    @Override
    public List<Member> findByRole(String role) {
        String sql = "SELECT * FROM member WHERE deleted = 0 AND role = ?";
        List<Member> list = new ArrayList<>();
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setString(1, role);
            try (ResultSet rs = ps.executeQuery()) {
                while (rs.next()) {
                    list.add(mapRow(rs));
                }
            }
        } catch (SQLException e) {
            throw new RuntimeException("依角色查詢失敗, role=" + role, e);
        }
        return list;
    }

    // ========== UPDATE ==========
    @Override
    public void update(Member member) {
        String sql = "UPDATE member SET account=?, password=?, name=?, email=?, phone=?, role=? "
                   + "WHERE id=? AND deleted=0";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setString(1, member.getAccount());
            ps.setString(2, member.getPassword());
            ps.setString(3, member.getName());
            ps.setString(4, member.getEmail());
            ps.setString(5, member.getPhone());
            ps.setString(6, member.getRole());
            ps.setInt(7, member.getId());
            ps.executeUpdate();
        } catch (SQLException e) {
            throw new RuntimeException("更新會員失敗, id=" + member.getId(), e);
        }
    }

    // ========== DELETE — 軟刪除 ==========
    @Override
    public void delete(int id) {
        String sql = "UPDATE member SET deleted = 1 WHERE id = ?";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            ps.setInt(1, id);
            ps.executeUpdate();
        } catch (SQLException e) {
            throw new RuntimeException("刪除會員失敗, id=" + id, e);
        }
    }

    // ========== COUNT — 總筆數 ==========
    @Override
    public long count() {
        String sql = "SELECT COUNT(*) FROM member WHERE deleted = 0";
        try (Connection conn = DBUtil.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql);
             ResultSet rs = ps.executeQuery()) {
            if (rs.next()) {
                return rs.getLong(1);
            }
        } catch (SQLException e) {
            throw new RuntimeException("計算會員總數失敗", e);
        }
        return 0;
    }

    // ========== mapRow — ResultSet 轉 JavaBean ==========
    private Member mapRow(ResultSet rs) throws SQLException {
        Member member = new Member();
        member.setId(rs.getInt("id"));
        member.setAccount(rs.getString("account"));
        member.setPassword(rs.getString("password"));
        member.setName(rs.getString("name"));
        member.setEmail(rs.getString("email"));
        member.setPhone(rs.getString("phone"));
        member.setRole(rs.getString("role"));
        member.setDeleted(rs.getBoolean("deleted"));
        return member;
    }
}
```

**各方法對照說明：**

| 方法 | SQL | 重點 |
|---|---|---|
| `insert()` | `INSERT INTO ... RETURN_GENERATED_KEYS` | `getGeneratedKeys()` 取得自增 ID |
| `findByAccount()` | `WHERE account = ? AND deleted = 0` | **登入驗證用**，回傳 Optional |
| `findById()` | `WHERE id = ? AND deleted = 0` | 編輯時查詢單筆 |
| `findAll()` | `SELECT * WHERE deleted = 0` | 列表頁使用 |
| `searchByName()` | `WHERE name LIKE ?` | `"%" + name + "%"` 模糊搜尋 |
| `findByRole()` | `WHERE role = ?` | 依角色篩選 |
| `update()` | `UPDATE ... WHERE id = ? AND deleted = 0` | 避免更新已刪除資料 |
| `delete()` | `UPDATE SET deleted = 1` | 軟刪除 |
| `mapRow()` | — | ResultSet → Member 對映 |

## 八、Controller 層：MemberServlet.java

```java
package com.example.servlet;

import com.example.dao.MemberDao;
import com.example.dao.impl.MemberDaoImpl;
import com.example.model.Member;

import jakarta.servlet.ServletException;
import jakarta.servlet.annotation.WebServlet;
import jakarta.servlet.http.HttpServlet;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import jakarta.servlet.http.HttpSession;

import java.io.IOException;
import java.util.List;
import java.util.Optional;

@WebServlet("/member/*")
public class MemberServlet extends HttpServlet {

    private final MemberDao memberDao = new MemberDaoImpl();

    // ========== GET 請求處理 ==========
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {

        String path = req.getPathInfo();
        if (path == null) {
            path = "/";
        }

        switch (path) {
            case "/", "/list" -> handleList(req, resp);
            case "/login"     -> handleLoginPage(req, resp);
            case "/logout"    -> handleLogout(req, resp);
            case "/add"       -> handleAddForm(req, resp);
            case "/edit"      -> handleEditForm(req, resp);
            case "/delete"    -> handleDelete(req, resp);
            case "/search"    -> handleSearch(req, resp);
            default           -> resp.sendError(HttpServletResponse.SC_NOT_FOUND, "找不到頁面: " + path);
        }
    }

    // ========== POST 請求處理 ==========
    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {

        req.setCharacterEncoding("UTF-8");

        String path = req.getPathInfo();
        if (path == null) {
            path = "/";
        }

        switch (path) {
            case "/login" -> handleLogin(req, resp);
            case "/save"  -> handleSave(req, resp);
            default       -> resp.sendError(HttpServletResponse.SC_NOT_FOUND, "找不到頁面: " + path);
        }
    }

    // ==================== 登入/登出 ====================

    // 顯示登入頁面
    private void handleLoginPage(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        req.getRequestDispatcher("/WEB-INF/views/login.jsp").forward(req, resp);
    }

    // 登入驗證
    private void handleLogin(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        String account = req.getParameter("account");
        String password = req.getParameter("password");

        // --- 後端驗證 ---
        if (account == null || account.trim().isEmpty()
                || password == null || password.trim().isEmpty()) {
            req.setAttribute("error", "帳號和密碼不可為空");
            req.getRequestDispatcher("/WEB-INF/views/login.jsp").forward(req, resp);
            return;
        }

        // --- 查詢帳號 ---
        Optional<Member> opt = memberDao.findByAccount(account.trim());
        if (opt.isEmpty()) {
            req.setAttribute("error", "帳號不存在");
            req.getRequestDispatcher("/WEB-INF/views/login.jsp").forward(req, resp);
            return;
        }

        Member member = opt.get();

        // --- 比對密碼 ---
        if (!member.getPassword().equals(password.trim())) {
            req.setAttribute("error", "密碼錯誤");
            req.getRequestDispatcher("/WEB-INF/views/login.jsp").forward(req, resp);
            return;
        }

        // --- 登入成功：將會員資訊存入 Session ---
        HttpSession session = req.getSession();
        session.setAttribute("loginMember", member);   // 整個 Member 物件存進 Session
        session.setAttribute("loginName", member.getName()); // 方便 JSP 顯示姓名

        // 重導向到會員列表
        resp.sendRedirect(req.getContextPath() + "/member/list");
    }

    // 登出
    private void handleLogout(HttpServletRequest req, HttpServletResponse resp)
            throws IOException {
        HttpSession session = req.getSession(false); // false = 不自動建立
        if (session != null) {
            session.invalidate();  // 銷毀整個 Session
        }
        resp.sendRedirect(req.getContextPath() + "/member/login");
    }

    // ==================== CRUD ====================

    // 列表查詢
    private void handleList(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        List<Member> members = memberDao.findAll();
        req.setAttribute("members", members);
        req.getRequestDispatcher("/WEB-INF/views/memberList.jsp").forward(req, resp);
    }

    // 新增表單
    private void handleAddForm(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        req.getRequestDispatcher("/WEB-INF/views/memberForm.jsp").forward(req, resp);
    }

    // 編輯表單
    private void handleEditForm(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        String idStr = req.getParameter("id");
        if (idStr == null || idStr.isEmpty()) {
            resp.sendRedirect(req.getContextPath() + "/member/list");
            return;
        }
        Optional<Member> opt = memberDao.findById(Integer.parseInt(idStr));
        if (opt.isPresent()) {
            req.setAttribute("member", opt.get());
            req.getRequestDispatcher("/WEB-INF/views/memberForm.jsp").forward(req, resp);
        } else {
            resp.sendRedirect(req.getContextPath() + "/member/list");
        }
    }

    // 刪除（軟刪除）
    private void handleDelete(HttpServletRequest req, HttpServletResponse resp)
            throws IOException {
        String idStr = req.getParameter("id");
        if (idStr != null && !idStr.isEmpty()) {
            memberDao.delete(Integer.parseInt(idStr));
        }
        resp.sendRedirect(req.getContextPath() + "/member/list");
    }

    // 搜尋
    private void handleSearch(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        String name = req.getParameter("name");
        String role = req.getParameter("role");
        List<Member> members;

        if (name != null && !name.trim().isEmpty()) {
            members = memberDao.searchByName(name.trim());
        } else if (role != null && !role.trim().isEmpty()) {
            members = memberDao.findByRole(role.trim());
        } else {
            members = memberDao.findAll();
        }

        req.setAttribute("members", members);
        req.getRequestDispatcher("/WEB-INF/views/memberList.jsp").forward(req, resp);
    }

    // 新增/編輯提交 (POST /member/save)
    private void handleSave(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {

        String idParam = req.getParameter("id");
        String account = req.getParameter("account");
        String password = req.getParameter("password");
        String name = req.getParameter("name");
        String email = req.getParameter("email");
        String phone = req.getParameter("phone");
        String role = req.getParameter("role");

        // --- 後端資料驗證 ---
        if (account == null || account.trim().isEmpty()) {
            req.setAttribute("error", "帳號不可為空");
            forwardToForm(req, resp, null);
            return;
        }
        if (password == null || password.trim().isEmpty()) {
            req.setAttribute("error", "密碼不可為空");
            forwardToForm(req, resp, null);
            return;
        }
        if (name == null || name.trim().isEmpty()) {
            req.setAttribute("error", "姓名不可為空");
            forwardToForm(req, resp, null);
            return;
        }
        if (role == null || role.trim().isEmpty()) {
            role = "user";
        }

        // --- 根據有無 id 決定新增或更新 ---
        if (idParam == null || idParam.isEmpty()) {
            Member member = new Member(account.trim(), password.trim(),
                    name.trim(), email, phone, role.trim());
            memberDao.insert(member);
        } else {
            Member member = new Member();
            member.setId(Integer.parseInt(idParam));
            member.setAccount(account.trim());
            member.setPassword(password.trim());
            member.setName(name.trim());
            member.setEmail(email);
            member.setPhone(phone);
            member.setRole(role.trim());
            memberDao.update(member);
        }

        resp.sendRedirect(req.getContextPath() + "/member/list");
    }

    // 轉發表單
    private void forwardToForm(HttpServletRequest req, HttpServletResponse resp,
                               Member member) throws ServletException, IOException {
        if (member != null) {
            req.setAttribute("member", member);
        }
        req.getRequestDispatcher("/WEB-INF/views/memberForm.jsp").forward(req, resp);
    }
}
```

### 登入流程圖解

```
瀏覽器 GET /member/login
  → handleLoginPage()
    → forward → login.jsp (空表單)

使用者輸入帳號密碼後送出
  → 瀏覽器 POST /member/login (account=admin&password=1234)
    → handleLogin()
      → 驗證帳號密碼非空
      → memberDao.findByAccount("admin")
        → SELECT * FROM member WHERE account='admin' AND deleted=0
        → Optional<Member>
      → 比對密碼: member.getPassword().equals("1234")
      → 登入成功:
        → session.setAttribute("loginMember", member)  ← 存入 Session
        → resp.sendRedirect(/member/list)
      → 登入失敗:
        → req.setAttribute("error", "密碼錯誤")
        → forward → login.jsp (顯示錯誤)
```

### Session 生命週期

```
使用者登入成功
  → HttpSession session = req.getSession()
  → session.setAttribute("loginMember", member)
  → Session ID 產生 (JSESSIONID cookie 存於瀏覽器)

使用者瀏覽其他頁面
  → 瀏覽器自動攜帶 JSESSIONID cookie
  → req.getSession() 取得同一個 Session
  → session.getAttribute("loginMember") → 取得登入會員

使用者登出
  → session.invalidate()  ← 銷毀 Session
  → 瀏覽器 JSESSIONID 失效
  → 之後請求被 LoginFilter 導向登入頁
```

### 10.2 LoginFilter.java — Session 保護（本專案新增）

```java
package com.example.filter;

import jakarta.servlet.*;
import jakarta.servlet.annotation.WebFilter;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import jakarta.servlet.http.HttpSession;
import java.io.IOException;

@WebFilter("/member/*")
public class LoginFilter implements Filter {

    // 不需要登入就能存取的路徑
    private static final String[] PUBLIC_PATHS = {"/login"};

    @Override
    public void doFilter(ServletRequest req, ServletResponse resp,
                         FilterChain chain) throws IOException, ServletException {

        HttpServletRequest request = (HttpServletRequest) req;
        HttpServletResponse response = (HttpServletResponse) resp;

        String path = request.getPathInfo();  // 取得 /member 之後的路徑
        if (path == null) {
            path = "/";
        }

        // --- 檢查是否為公開路徑 ---
        for (String publicPath : PUBLIC_PATHS) {
            if (path.equals(publicPath)) {
                chain.doFilter(req, resp);  // 公開路徑，直接放行
                return;
            }
        }

        // --- 檢查 Session 是否有登入資訊 ---
        HttpSession session = request.getSession(false); // false = 不自動建立
        if (session != null && session.getAttribute("loginMember") != null) {
            chain.doFilter(req, resp);  // 已登入，放行
        } else {
            // 未登入，導向登入頁
            response.sendRedirect(request.getContextPath() + "/member/login");
        }
    }
}
```

**LoginFilter 運作流程：**

```
瀏覽器請求 /member/xxx
    │
    ▼
LoginFilter.doFilter()
    │
    ├── 路徑是 /member/login？
    │     YES → chain.doFilter() → 放行 → MemberServlet 處理
    │
    ├── Session 有 loginMember？
    │     YES → chain.doFilter() → 放行 → MemberServlet 處理
    │
    └── 以上皆非（未登入）
          → sendRedirect → /member/login
```

**為什麼 LoginFilter 要設定 `PUBLIC_PATHS`？**
如果不排除 `/member/login`，會造成**無限迴圈**：
```
未登入 → 導向 /member/login
→ Filter 攔截 /member/login → 還是未登入 → 導向 /member/login
→ 無限迴圈...
```

### 10.3 Filter 執行順序

```
請求進入
    │
    ▼
EncodingFilter (設定 UTF-8 編碼)
    │  chain.doFilter()
    ▼
LoginFilter (檢查 Session 登入)
    │  chain.doFilter()
    ▼
MemberServlet (處理業務邏輯)
    │
    ▼
回應送出
```

**兩個 Filter 的職責：**
| Filter | URL 匹配 | 職責 |
|---|---|---|
| `EncodingFilter` | `/*`（所有請求） | 統一設定請求/回應編碼為 UTF-8 |
| `LoginFilter` | `/member/*`（會員相關請求） | 檢查 Session，未登入則導向登入頁 |

## 十三、Session 機制完整說明

### 什麼是 Session？

```
使用者 A (瀏覽器)  ────── JSESSIONID=abc123 ──────→  伺服器 Session 空間
使用者 B (瀏覽器)  ────── JSESSIONID=def456 ──────→  伺服器 Session 空間
```

- **Session** 是伺服器端的記憶體空間，每個使用者有獨立的一份
- **JSESSIONID** 是 Session 的唯一識別碼，存於 Cookie 中
- 瀏覽器每次請求會自動攜帶 JSESSIONID Cookie，伺服器據此找到對應的 Session

### Session 常用 API

| API | 說明 |
|---|---|
| `req.getSession()` | 取得 Session，若不存在則自動建立 |
| `req.getSession(false)` | 取得 Session，若不存在則回傳 null |
| `session.setAttribute("key", value)` | 將資料存入 Session |
| `session.getAttribute("key")` | 從 Session 取出資料 |
| `session.removeAttribute("key")` | 從 Session 移除資料 |
| `session.invalidate()` | 銷毀整個 Session（登出時使用） |
| `session.getId()` | 取得 Session ID (JSESSIONID) |

### Session 預設逾時

- Tomcat 預設 **30 分鐘** 無活動自動過期
- 可在 `web.xml` 設定：
```xml
<session-config>
    <session-timeout>30</session-timeout>  <!-- 單位：分鐘 -->
</session-config>
```

## 十四、常見錯誤排解

| 錯誤現象 | 原因 | 解決方式 |
|---|---|---|
| 登入後立刻被導回登入頁 | Session 未正確建立 | 確認 `req.getSession()` 在密碼比對成功後才呼叫 |
| 所有頁面都跳到登入頁 | LoginFilter 未排除 login 路徑 | 確認 `PUBLIC_PATHS` 包含 `"/login"` |
| 登出後按瀏覽器返回仍能存取 | 瀏覽器快取 | 回應加入 `Cache-Control: no-cache` 標頭 |
| `JSESSIONID` 每次請求都不同 | Cookie 未啟用或 domain 設定錯誤 | 確認瀏覽器 Cookie 已啟用 |
| `Session is new` 始終為 true | Session 建立後未正確回傳 Cookie | 確認未在 `getSession()` 前呼叫 `response.sendRedirect()` |
| 帳號重複仍可新增 | DB 未設定 UNIQUE 約束 | 確認 `account` 欄位有 `UNIQUE` 約束 |
| 登入帳號正確但密碼比對失敗 | 密碼前後有空白字元 | 使用 `.trim()` 移除空白再比對 |
| 中文姓名顯示亂碼 | 編碼未設定 | 確認 EncodingFilter 存在且執行順序在 LoginFilter 之前 |
| `Table 'practice_db.member' doesn't exist` | 資料表未建立 | 執行第三節 SQL 建表腳本 |
